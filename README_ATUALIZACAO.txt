OPÇÃO 1: COMECE IMEDIATAMENTE (3 minutos)

Windows:
  1. winget install astral-sh.uv
  2. uv sync
  3. uv run python examples/basic_example.py

macOS:
  1. brew install uv
  2. uv sync
  3. uv run python examples/basic_example.py

Linux:
  1. curl -LsSf https://astral.sh/uv/install.sh | sh
  2. uv sync
  3. uv run python examples/basic_example.py
